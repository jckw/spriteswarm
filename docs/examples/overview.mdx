## Quick Start

1. Copy an example YAML file
2. Update the `sprite.name` to match your Sprite in Sprites.dev
3. Customize the `run` command for your use case
4. Upload to SpriteSwarm:

```bash
curl -X POST https://your-app.fly.dev/admin/automations \
  -H "X-Admin-Token: your-token" \
  -H "Content-Type: text/yaml" \
  --data-binary @your-automation.yaml
```

## Building Your Own

Every automation follows the same pattern:

```yaml
id: unique-identifier

sprite:
  name: your-sprite-name

source:
  type: webhook  # or cron
  event: some.event

match:
  - payload.field == "value"  # optional filters

run: |
  Your command with {{payload.variables}}
```

See [Writing Automations](/automations/overview) for the full reference.
